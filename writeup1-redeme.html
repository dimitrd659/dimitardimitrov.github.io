HackTheBox • Redis Exploitation Writeup

Task 1 – Open Port

Which TCP port is open on the machine?
Answer: 6379

Initial port scan:

nmap -p- -sV 10.129.70.193

Task 2 – Running Service

Which service is running on the open port?
Answer: redis

Task 3 – Database Type

What type of database is Redis?
Answer: In-memory Database

Task 4 – CLI Utility

Command-line utility to interact with Redis server?
Answer: redis-cli

Task 5 – Host Flag

Which flag specifies the hostname?
Answer: -h

redis-cli --help

Task 6 – Server Information

Command to get Redis server info and statistics?
Answer: info

Task 7 – Redis Version

Redis server version on the target?
Answer: 5.0.7

Task 8 – Select Database

Command to switch to a specific database?
Answer: select

Task 9 – Number of Keys in DB 0

How many keys in database index 0?
Answer: 4

select 0
dbsize

Task 10 – List All Keys

Command to list all keys in current database?
Answer: keys *

Task 11 – Root Flag

Submit root flag
Flag: 03e1d2b376c37ab3f5319922053953eb

After running keys * we see the key flag and retrieve it with:

get flag
